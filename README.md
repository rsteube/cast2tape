# cast2tape

Experimental converter from [asciinema](https://github.com/asciinema/asciinema) `cast` file recorded with `--stdin` to a [vhs](https://github.com/charmbracelet/vhs) `tape` file.


```sh
./cast2tape ../example.cast 
Set Shell elvish
Type "echo test"
Backspace 4
Type "example"
Enter
Type "vhs "
Tab 7
Enter 2
```
